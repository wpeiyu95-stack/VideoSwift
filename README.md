# VideoSwift

VideoSwift 是一个 AI 视频速读与总结工具。

它的目标是帮助用户把冗长的视频内容，快速转化为清晰、结构化的摘要，让用户用更短时间理解视频核心信息。

## 项目定位

很多视频内容很长，例如课程、会议、访谈、讲座、产品演示等。

VideoSwift 希望解决的问题是：

- 不想完整看完长视频
- 想快速知道视频讲了什么
- 想提取核心观点和行动清单
- 想把视频内容转成可复习、可整理的文字资料

## 当前版本

当前版本是 VideoSwift 的前端原型版本。

已完成功能：

- 首页展示

当前还没有接入真实 AI API，也没有真实解析视频字幕。

## API Key 安全说明

为了避免真实密钥被提交到 GitHub，请将本地环境变量写入 `.env` 文件，并确保不要把真实值写进仓库。

- 仓库已通过 `.gitignore` 忽略 `.env`
- `.env.example` 只提供示例值，不包含任何真实密钥
- 开发时如需配置密钥，请参考 `.env.example` 并在本地单独创建 `.env`
- 当前版本不会调用真实 OpenAI API
